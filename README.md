# Programozas-II
 A Neumann János Egyetem mérnökinformatikus BSc hallgatói számára készült órai anyagok a Programozás II. című tárgyhoz!

## 1. labor
 Az első labor keretein belül átismételtük, hogy mit tanultunk az előzmény tárgy során. Megoldottunk egy stringekkel kapcsolatos feladatot, valamint egy ZH-hoz haosnlító példát ismétlésképpen. Gyakorláshoz pedig feltöltöttem az 1. labor mappába egy gyakorlas.pdf-et, valamint az ahhoz szükséges forrásállományt is. Aki elkészíti a feladatot, hozza magával a következő gyakorlatra, és jelezze felém, hogy készült házi feladattal.
